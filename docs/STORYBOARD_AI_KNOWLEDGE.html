Description

Summary.

  • 호텔 AI 어시스턴트(컨시어지)가 응답 시 참고하는 지식 기반(RAG) 문서를 관리하는 화면
  • 호텔 운영 정책, 관광 가이드, 시설 안내 등의 데이터를 추가/수정/삭제 가능
  • 현재 AI 엔진의 동작 상태 및 학습(동기화) 현황을 한눈에 파악할 수 있음
1

+ 지식 정보 추가 (Add Knowledge)

· 새로운 호텔 규정, 이벤트, 메뉴얼 등을 AI가 학습하도록 텍스트나 파일(PDF 등) 형태로 추가하는 진입점
2

지식 라이브러리 (Knowledge Library)

· 현재 AI가 학습하여 알고 있는 문서 리스트

· 카테고리 태그(체크인/아웃, 부대시설 등)와 제목, 상세 설명 노출

· 최종 업데이트 일자를 명시하여 데이터의 최신성 관리
3

AI 엔진 상태 (Engine Status)

· 정상 작동 중: AI 모델 서버(Ollama/Python)의 Health 상태 표시

· 총 지식 문서 수: RAG 벡터 DB에 적재된 총 문서량

· 동기화 주기: VectorDB가 최신화된 시간

· 학습 정확도: 고객 질문 대비 적절한 문서를 찾은 비율(Hit Rate 등)
4

시스템 카테고리 (Category Filter)

E1 지식 정보 등록/수정
