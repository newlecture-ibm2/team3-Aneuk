Description

Summary.

  • 프론트 데스크 미처리/예외 요청 대시보드 및 상세 채팅 모달(A-02-M)
  • 상태별 현황 제공 및 가장 시급한 예외 건 우선 노출
  • 빠른 고객 응대를 위한 실시간 채팅(상담) 진입 지원
1

상단 현황 요약 (Status Cards)

· 대기 중: AI 처리가 불가하여 직원 개입이 필요한 건

· 진행 중: 직원이 배정되어 처리 중인 건
2

섹션 타이틀 & 배지

우선순위가 가장 높은 "미처리 / 예외 요청"을 표시하며 붉은색 배지로 경각심 부여
3

요청 리스트 정보

· AI 요약 제목 및 연회색 박스에 고객 발화 원문 동시 제공
4

상담 개입 액션

· 상담 시작: 아래의 5번 화면(모달창)을 열어 상세 대화 확인
5

상담 상세 모달 (A-02-M)

· 고객과 AI의 대화 내역(히스토리) 확인

· 하단 입력창을 통해 직원이 직접 텍스트 입력 시 실시간 고객 응대 가능
🎯 Event Description
E1 상담 모달 진입 (상담 시작)

· 트리거: 특정 카드의 '상담 시작' 버튼 클릭 (4번)

· 동작: 배경 Dim 처리 후 채팅 상세 히스토리 모달(5번 화면) 노출

· API: GET /api/staff/requests/{id}/chat
E2 요청 취소 (무시하기)

· 트리거: '무시하기' 버튼 클릭

· 동작: 리스트 항목 숨김 처리 및 상태 DB 업데이트

Check Point

WebSocket 연동을 통한 리스트 실시간 업데이트 필수

직원이 모달에서 직접 메시지 발송 시, 해당 세션의 AI 자동 응대는 일시 중지됨
