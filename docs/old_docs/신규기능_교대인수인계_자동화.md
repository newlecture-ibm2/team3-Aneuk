# [신규 기능 제안] 스마트 업무 일지 및 교대 인수인계 자동화

## 1. 개요
호텔 현장에서 교대 시 구두나 수기로 진행하던 인수인계 과정을 시스템 내에서 자동화합니다. AI가 고객의 요청 및 업무 처리 내역 중 '미완료된 잔여 업무'와 '관리자 개입이 필요했던 특이사항'을 자동으로 취합하고 분석하여, 다음 교대 근무자에게 꼭 필요한 핵심 내용만 자연어로 요약 브리핑해 주는 기능입니다.

## 2. 해결하고자 하는 문제 (Pain-point)
*   **데이터 누락 및 휴먼 에러**: 바쁜 교대 시간에 구두나 수기로 전달하다 보니 중요한 인계 사항(예: 특정 객실의 지연된 수리 건)이 누락되는 경우가 잦음.
*   **파편화된 시스템**: 직원들이 시스템 상의 태스크를 확인하는 업무와, 별도의 엑셀/수기 인수인계 대장을 작성하는 이중 업무를 수행해야 함.

## 3. 핵심 기능 상세 (MVP Scope)

### ① 인수인계 대상 태스크 자동 취합 (Auto-Sync)
*   관리자 및 직원 대시보드 내에 **'교대 인수인계' 탭**을 제공합니다.
*   해당 교대 시간(Shift) 내에 발생한 태스크들을 백엔드에서 필터링하여 다음 세 가지 그룹으로 취합합니다.
    1. 아직 완료되지 않은 잔여 업무 (`PENDING`, `IN_PROGRESS`)
    2. 처리는 완료되었으나 **관리자 컨펌/에스컬레이션을 거친 특이사항** (`confidence < 0.7` 또는 `ESCALATED` 이력)
    3. 정상적으로 자동 분배되어 완료된 업무 중 **주요도(Priority)가 높은 핵심 태스크** (`URGENT`, `HIGH`)
*   직원이 퇴근 전 별도로 데이터를 입력할 필요 없이 시스템의 실시간 업무 상태와 예외 처리 내역이 100% 연동됩니다 (Zero-Data-Entry).

### ② AI 자동 브리핑 요약 (AI Handover Summary)
*   취합된 통합 태스크 데이터(JSON)를 바탕으로, AI(Gemini)가 교대 근무자가 즉시 파악해야 할 **핵심 사항(잔여 업무 + 특이사항 + 주요 처리 내역)을 3~5줄로 자동 요약**하여 화면 상단에 띄워줍니다.
*   *예시: "302호 전구 교체 건이 3시간째 미처리 상태로 우선 확인 필요합니다. **[주요 처리]** 금일 VIP 501호 누수 긴급(URGENT) 건은 시설팀에서 10분 만에 조치 완료했습니다. **[특이사항]** 405호에서 규정에 없는 가습기 요청이 있어 프론트 창고 확인 후 예외 대여 처리했습니다."*

## 4. 확장 계획 (Phase 2)
초기 서비스 안정화 이후, 본 기능은 완벽한 양방향 '내부 스프레드시트' 형태로 진화합니다.

### ① 동적 컬럼 매핑 (Dynamic Column Mapping) 도입
*   **완벽한 역방향 Export 지원**: 호텔이 기존에 사용하던 엑셀 양식의 헤더를 시스템 필드와 최초 1회 매핑해 두면, 시스템 데이터를 Export(내보내기)할 때 호텔의 고유 엑셀 양식(원래 쓰던 헤더명과 순서) 그대로 복원하여 출력할 수 있습니다. 
*   단순 AI 파싱이 아닌 동적 매핑 방식을 채택하여 양방향 호환성과 엔터프라이즈 데이터의 무결성을 보장합니다.

### ② 정적 루틴(Routine) 관리
*   매일 고정적으로 해야 하는 체크리스트(마감 청소 확인, 비품 창고 재고 파악 등)를 템플릿화하여 그리드에 손쉽게 불러오는 기능을 제공합니다.

## 5. 기대 효과 (비즈니스 임팩트)
*   **완결된 업무 루프 시연**: `고객 요청 → AI 분석 → 직원 처리 → (잔여 업무 + 특이사항) AI 인수인계 브리핑`으로 이어지는 하루 단위의 완벽한 호텔 운영 사이클을 제시할 수 있습니다.
*   **즉각적인 실무 효율 극대화**: 직원이 퇴근 전 수기로 인수인계 사항을 고민하고 정리하던 시간을 '0분'으로 단축하여, 현장 도입 거부감을 대폭 낮춥니다.
