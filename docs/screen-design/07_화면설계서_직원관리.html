직원 추가

✨ UI 뷰: 직원 추가 팝업

리스트 우측 상단의 '+ 직원 추가' 버튼을 클릭하면 화면 전체가 어두워지며 중앙에 나타나는 모달(Modal) 팝업입니다.

메인 화면 위에 독립된 레이어로 배치되며, 모달 바깥 영역을 클릭하거나 취소 버튼을 누르면 팝업이 닫힙니다.

Description
1설정 메뉴· 사이드바 하단 '설정' 그룹에 부서 관리, 객실 관리, 직원 관리 위치
· 관리자 전용 메뉴 (ADMIN role만 접근 가능)
2+ 직원 추가 (모달)· '직원 추가' 버튼 클릭 시 직원 정보 입력 모달 팝업
· 직원 가입 페이지는 따로 존재하지 않음 (Admin-Provisioning 원칙)
· 이름, 부서, 역할 입력 후 저장 시 PIN 자동 발급
3직원 리스트· 이름: 실명
· 부서: 배지 형태의 소속 부서 태그
· 역할: 직급/포지션 (팀장, 매니저, 셰프 등)
· PIN 번호: 로그인에 사용되는 개인 고유 PIN
· 상태: ● 활성 / ● 비활성 (퇴사자)
· 액션(✏️/🗑️): 수정(정보 편집, 비활성화) 및 삭제
🎯 Event
E1직원 추가
· API: POST /admin/staff
· 이름 + 부서 + 역할 입력 → PIN 자동 생성
E2직원 수정/비활성화
· API: PUT /admin/staff/{id} / PATCH /admin/staff/{id}/status
· 비활성화 즉시 해당 PIN 로그인 불가 + 기존 세션 강제 만료
💡 Check Point
C1PIN 하나로 역할 자동 판별 (통합 로그인 UI-LOGIN 참고)
C21코드 1기기: 동일 PIN 새 기기 접속 시 기존 세션 강제 만료
