Description
1호텔 로고 + 타이틀· '호텔 운영 관리 시스템' 서브 타이틀
· 관리자/직원 구분 없는 단일 로그인 화면
2PIN 코드 입력· 관리자와 직원 모두 동일한 화면에서 PIN 입력
· 입력된 PIN에 따라 서버가 역할(ADMIN/STAFF) 자동 판별
· 개별 칸 자동 포커스 이동, ● 마스킹
3로그인 버튼· 6자리 입력 완료 시 자동 제출 또는 버튼 클릭
· 서버가 PIN → 역할 판별 → JWT 발급(role, deptCode 포함)
· ADMIN: 관리자 대시보드(A-02)로 이동
· STAFF: 소속 부서 요청 대시보드(S-02)로 이동
🎯 Event
E1통합 PIN 로그인
· API: POST /auth/login (단일 엔드포인트)
· 서버가 PIN으로 DB 조회 → role(ADMIN/STAFF) 자동 판별
· JWT payload: { role, deptCode, staffId, name }
· 프론트: role에 따라 관리자/직원 메인 페이지로 자동 라우팅
💡 Check Point
C1로그인 화면은 하나, PIN 번호가 역할을 결정 (관리자/직원 구분 UI 없음)
C21코드 1기기 원칙: 동일 PIN 새 기기 접속 시 기존 세션 강제 만료
C3비활성화(Blocked) 계정: "비활성화된 계정입니다. 관리자에게 문의하세요"
