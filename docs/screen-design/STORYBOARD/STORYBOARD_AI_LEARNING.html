Summary.

  • 과거 고객과의 상담 로그를 모니터링하고 AI의 응답 품질을 관리하는 화면
  • 의미 있는 고객 질의 및 AI의 우수 응답을 선별하여 '지식(RAG)'으로 재학습시키는 기능 제공
  • [인터랙션] 좌측 화면의 첫 번째 카드를 클릭해 보세요! 상세 모달창이 나타납니다.
1

상담 로그 리스트 항목 (Clickable)

· 고객이 남긴 텍스트 원문(우측 정렬된 블랙 버블)을 프리뷰 형태로 제공

· 해당 카드를 클릭 시, AI가 어떻게 답변했는지 확인할 수 있는 상세 모달 팝업 오픈
2

로그 데이터 상세 팝업 (Modal)

· 고객의 발화(블랙)와 AI의 실제 답변(화이트)이 대화형 UI로 출력됨

· AI가 환각(Hallucination) 없이 부서를 잘 안내했는지 관리자가 검수 가능
3

학습 데이터 처리 액션

· 지식으로 저장: 해당 응답 세트를 우수 사례로 판단하여 Vector DB에 저장 (추후 유사 질문 시 참고)

· 제외: 단순 변심, 욕설, 무의미한 로그이거나 AI 답변이 부적절한 경우 학습에서 제외 처리
🎯 Event Description
E1 카드 클릭 시 모달창 팝업

· 트리거: 로그 리스트 항목 클릭

· 동작: 백그라운드가 딤 처리되며 대화 내용이 담긴 모달 오픈
E2 지식으로 저장 처리

· 트리거: '지식으로 저장' 버튼 클릭

· 동작: 해당 Q&A 페어를 텍스트 파일이나 임베딩 데이터로 변환 후 AI 지식 라이브러리에 반영
