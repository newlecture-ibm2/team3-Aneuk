객실 추가

✨ UI 뷰: 객실 추가 팝업

리스트 우측 상단의 '+ 객실 추가' 버튼을 클릭하면 화면 전체가 어두워지며 중앙에 나타나는 모달(Modal) 팝업입니다.

메인 화면 위에 독립된 레이어로 배치되며, 모달 바깥 영역을 클릭하거나 취소 버튼을 누르면 팝업이 닫힙니다.

Description
1설정 메뉴
· 사이드바 하단 '설정' 그룹에 위치
2+ 객실 추가 (모달)
· 시스템에 새로운 객실 마스터 데이터를 등록하는 팝업
3객실 리스트
· 객실의 호수, 층, 타입(STANDARD/DELUXE/SUITE)을 관리
🎯 Event
E1객실 추가 저장
· 모달에서 호수/층/타입 입력 후 저장 클릭
· API: POST `/admin/rooms`
E2수정 / 삭제
· ✏️ 클릭 시 수정 모달 오픈, 🗑️ 클릭 시 삭제 확인 후 제거
· API: PUT/DELETE `/admin/rooms/{id}`
💡 Check Point
C1객실 번호(호수)는 중복 등록 불가, 숫자만 허용
C2투숙 중인 객실은 삭제/수정이 제한됨 (Hard Delete 정책 적용)
