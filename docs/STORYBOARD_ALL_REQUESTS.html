Description

Summary.

  • 모든 부서(하우스키핑, 시설, F&B, 프론트 등)의 요청을 통합 관리하는 대시보드
  • 호텔 전체의 오퍼레이션 현황을 한눈에 파악할 수 있는 마스터 뷰(Master View)
  • 특정 부서 화면과 동일한 칸반 인터페이스 제공
1

전체 요청 (All Requests) 뷰

· 호텔 내에서 발생하는 모든 카테고리의 고객/직원 요청이 모이는 메인 허브
2

통합 칸반 보드 (Columns)

· 대기 중, 진행 중, 완료됨으로 상태를 나누어 전체 태스크의 병목 현상 파악 용이
3

우선순위 (Priority) 뱃지

· HIGH, MEDIUM, LOW로 표기되어, 관리자가 여러 부서 요청 중 가장 먼저 처리해야 할 건을 식별 가능
4

다양한 카테고리 요청 혼재

· 시설(에어컨 수리), 하우스키핑(수건 교체), F&B(룸서비스) 등 여러 부서의 카드가 한 곳에 나열됨

· 필요 시 상단 필터를 통해 부서별로 필터링 가능
5

긴급/특수 상황 관리

· '객실 내 응급 환자' 등 HIGH 우선순위의 프론트 데스크 개입 필요 건도 통합 보드에서 모니터링 가능

Check Point

타 부서(예: 하우스키핑) 화면에서 특정 건이 '진행 중'으로 변경되면, 전체 뷰에서도 실시간 동기화되어야 함

총지배인(GM)이나 마스터 관리자가 주로 사용하는 화면으로, 권한 분리 철저히 확인
